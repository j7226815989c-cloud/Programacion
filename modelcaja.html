Modelo de Caja CSS

El modelo de caja describe cómo cada elemento HTML se representa como una caja con contenido, padding, border y margin.

Code
Explicación

El modelo de caja CSS es esencialmente una caja que envuelve cada elemento HTML.

Consta de: contenido, padding, border y margin.

Este texto es el contenido del cuadro.
